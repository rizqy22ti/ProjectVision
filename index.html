Peringatan Keamanan Browser (CORS) Terdeteksi!

Anda membuka halaman ini langsung dari file lokal (`file://`). Browser memblokir pengunduhan file model kecerdasan buatan (`model.json`) secara lokal karena alasan keamanan.

Silakan jalankan server lokal sederhana untuk menggunakan aplikasi ini:

python -m http.server 5000

Setelah menjalankan perintah tersebut di terminal proyek ini, buka alamat http://localhost:5000 di browser Anda.
